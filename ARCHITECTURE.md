# Почтовая рассылка расчетных листов ФитоФарм

Приложение для автоматической рассылки писем с вложениями через Outlook.

## Структура проекта

```
c:\VS Code/
├── main.py              # Точка входа в приложение
├── requirements.txt     # Зависимости Python
├── .gitignore          # Git ignore файл
│
├── backend/            # Бизнес-логика
│   ├── __init__.py
│   ├── email_service.py      # Сервис отправки email
│   ├── file_service.py       # Сервис работы с файлами
│   ├── excel_service.py      # Сервис чтения Excel
│   └── settings_manager.py   # Менеджер настроек
│
├── frontend/           # Пользовательский интерфейс
│   ├── __init__.py
│   ├── app.py                # Главный класс приложения
│   └── ui_components.py      # UI компоненты
│
├── core/             # Базовые компоненты
│   ├── __init__.py
│   ├── constants.py          # Константы приложения
│   └── logger_config.py      # Настройка логирования
│
└── models/           # Модели данных
    ├── __init__.py
    └── email_data.py         # Модели: EmailRecipient, EmailConfig, SendResult
```

## Архитектура

Приложение построено по принципу **разделения ответственности**:

- **Backend** — бизнес-логика (отправка писем, работа с файлами, настройки)
- **Frontend** — пользовательский интерфейс (Tkinter GUI)
- **Core** — базовые утилиты (логгер, константы)
- **Models** — модели данных (dataclasses)

### Преимущества новой архитектуры

1. **Разделение ответственности** — каждый модуль отвечает за свою задачу
2. **Тестируемость** — бизнес-логика изолирована от UI
3. **Расширяемость** — легко добавить новые функции
4. **Читаемость** — код структурирован и понятен

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите приложение:
```bash
python main.py
```

## Требования

- Python 3.9+
- Windows (для работы с Outlook через COM)
- pandas
- pywin32

## Использование

1. Выберите аккаунт Outlook
2. Укажите путь к Excel файлу с данными
3. Укажите пути к папкам с файлами
4. Настройте тему и текст письма
5. Нажмите "Начать рассылку"

## Формат Excel файла

Файл должен содержать колонки:
- `email` — адрес получателя
- `файл_01` — имя файла из папки 1
- `файл_02` — имя файла из папки 2
- `файл_03` — имя файла из папки 3
