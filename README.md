# EmailSender - Почтовая рассылка расчетных листов ФитоФарм

Приложение для автоматической рассылки писем с вложениями через **Microsoft Outlook** или **SMTP сервер**.

## 🚀 Быстрый старт

### Запуск из исходного кода

```bash
# 1. Установите зависимости
python -m pip install -r requirements.txt

# 2. Запустите приложение
python main.py
```

### Запуск готового EXE файла

```bash
# EXE файл находится в папке dist
dist\EmailSender.exe
```

## 📋 Требования

- **Python 3.9+** (для запуска из исходного кода)
- **Windows 7+** (требуется для работы с Outlook через COM)
- **Microsoft Outlook** (обязательно для режима Outlook, опционально для SMTP)

## 📦 Установка зависимостей

```bash
python -m pip install -r requirements.txt
```

**Зависимости:**
- `pandas>=2.0.0` — работа с Excel файлами
- `pywin32>=305` — COM автоматизация Outlook

## 🔨 Сборка в EXE файл

```bash
# Быстрая сборка (Windows)
build.bat

# Или вручную
python -m pip install pyinstaller
pyinstaller --clean email_sender.spec
```

После сборки EXE файл будет в папке `dist\EmailSender.exe` (~35 МБ).

## 📖 Использование

1. **Запустите приложение** (`python main.py` или `dist\EmailSender.exe`)

2. **Выберите режим отправки:**
   - **Outlook (COM)** — отправка через установленный Microsoft Outlook
   - **SMTP** — прямая отправка через SMTP сервер (Yandex, Gmail и др.)

3. **Для режима Outlook:** выберите аккаунт из выпадающего списка

4. **Для режима SMTP:** нажмите «⚙️ Настройки SMTP» и укажите:
   - SMTP сервер и порт
   - Логин и пароль
   - При необходимости — SSL/TLS

5. **Укажите путь к Excel файлу** со списком получателей

6. **Укажите пути к папкам:**
   - Папка с расчетными листами
   - Папка с реестрами выдачи
   - Папка с дополнительными файлами

7. **Настройте письмо:**
   - Тема автоматически генерируется с прошлым месяцем (например, "Расчетные листы Февраль 2026")
   - Текст письма можно отредактировать

8. **Нажмите "Начать рассылку"**

9. **Отслеживайте прогресс** в окне логов

10. **При ошибках отправки** — нажмите кнопку «📋 Ошибки» для просмотра списка e-mail адресов с неудачной отправкой

## 📊 Формат Excel файла

Excel файл должен содержать следующие колонки:

| Колонка | Описание |
|---------|----------|
| `email` | Адрес получателя (обязательно) |
| `файл_01` | Имя файла из папки 1 (расчетные листы) |
| `файл_02` | Имя файла из папки 2 (реестры выдачи) |
| `файл_03` | Имя файла из папки 3 (доп. файлы) |

**Пример:**

| email | файл_01 | файл_02 | файл_03 |
|-------|---------|---------|---------|
| ivanov@company.com | Иванов_ЛС.pdf | Реестр_001.xlsx | |
| petrov@company.com | Петров_ЛС.pdf | Реестр_001.xlsx | Премия.xlsx |

## ⚙️ Настройки

Настройки автоматически сохраняются в файл `settings.json` при закрытии приложения:

- Пути к Excel файлу и папкам
- Аккаунт Outlook / SMTP настройки
- Количество потоков
- Текст письма
- Режим отправки (Outlook/SMTP)

**Тема письма НЕ сохраняется** — генерируется автоматически при каждом запуске (прошлый месяц).

## 🔧 Возможности

- ✅ **Два режима отправки** — Outlook (COM) или SMTP (прямая отправка)
- ✅ **Многопоточная рассылка** (настраиваемое количество потоков, рекомендуется 3-5)
- ✅ **Автоматическая тема письма** с прошлым месяцем
- ✅ **Прикрепление файлов** из трёх разных папок
- ✅ **Предварительный просмотр** письма перед рассылкой
- ✅ **Пауза/продолжение** рассылки
- ✅ **Отмена рассылки** в любой момент
- ✅ **Сохранение настроек** между запусками
- ✅ **Логирование** всех операций
- ✅ **Копирование текста** из лога (Ctrl+C, Ctrl+Shift+C)
- ✅ **Экспорт ошибок** — сохранение e-mail адресов с ошибками в текстовый файл
- ✅ **Настройка SMTP** — поддержка SSL/TLS, настраиваемый порт

## ⚠️ Важные замечания

1. **Режим Outlook** — требуется установленный Microsoft Outlook
2. **Режим SMTP** — работает без Outlook, требуется доступ к SMTP серверу
3. **Только Windows** — используется COM автоматизация (для режима Outlook)
4. **Первый запуск** может занять больше времени (инициализация Outlook)
5. **Антивирус** может блокировать EXE файл (добавьте в исключения)

## 📞 Поддержка

При возникновении проблем:
1. Проверьте, что Outlook установлен и настроен (для режима Outlook)
2. Проверьте настройки SMTP сервера (для режима SMTP)
3. Убедитесь, что Excel файл содержит требуемые колонки
4. Проверьте логи в файле `email_sender.log`
5. Используйте кнопку «📋 Ошибки» для просмотра адресов с неудачной отправкой
