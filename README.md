# EmailSender — Почтовая рассылка расчётных листов «ФитоФарм»

Приложение для автоматической рассылки писем с вложениями через **Microsoft Outlook (COM)** или **SMTP** (Mail.ru, Yandex, Gmail и др.).

Поддерживает многопоточную отправку с «разогревом», пакетными паузами и защитой от спам-фильтров.

---

## 🚀 Быстрый старт

### Запуск из исходного кода

```bash
# 1. Установите зависимости
python -m pip install -r requirements.txt

# 2. Запустите приложение
python main.py
```

### Запуск готового EXE-файла

```bash
# EXE-файл находится в папке dist
dist\EmailSender.exe
```

---

## 📋 Требования

- **Python 3.9+** (для запуска из исходного кода)
- **Windows 7+** (требуется для работы с Outlook через COM)
- **Microsoft Outlook** (обязателен только для режима Outlook COM)
- **SMTP-сервер** (для режима SMTP: Mail.ru, Yandex, Gmail и др.)

## 📖 Использование

### 1. Запустите приложение

```bash
python main.py
# или
dist\EmailSender.exe
```

### 2. Выберите режим отправки

| Режим | Описание |
|-------|----------|
| **Outlook (COM)** | Отправка через установленный Microsoft Outlook |
| **SMTP** | Прямая отправка через SMTP-сервер (Yandex, Gmail и др.) |

### 3. Настройте параметры

#### Для режима Outlook:
- Выберите аккаунт из выпадающего списка

#### Для режима SMTP:
- Нажмите **⚙️ Настройки SMTP** и укажите:
  - SMTP-сервер и порт
  - Логин и пароль
  - SSL/TLS (при необходимости)
  - Имя отправителя

### 4. Укажите пути к файлам и папкам

- **Excel-файл** со списком получателей
- **Папка 1** — расчётные листы
- **Папка 2** — реестры выдачи
- **Папка 3** — дополнительные файлы

### 5. Настройте письмо

- **Тема** — автоматически генерируется с прошлым месяцем (например, «Расчётные листы Февраль 2026»)
- **Текст письма** — можно отредактировать

### 6. Начните рассылку

- Нажмите **«Начать рассылку»**
- Отслеживайте прогресс в окне логов
- При необходимости используйте **Паузу** или **Отмену**

## 📊 Формат Excel-файла

Excel-файл должен содержать следующие колонки:

| Колонка | Описание |
|---------|----------|
| `email` | Адрес получателя (обязательно) |
| `файл_01` | Имя файла из папки 1 (расчётные листы) |
| `файл_02` | Имя файла из папки 2 (реестры выдачи) |
| `файл_03` | Имя файла из папки 3 (доп. файлы) |

**Пример:**

| email | файл_01 | файл_02 | файл_03 |
|-------|---------|---------|---------|
| ivanov@company.com | Иванов_ЛС.pdf | Реестр_001.xlsx | |
| petrov@company.com | Петров_ЛС.pdf | Реестр_001.xlsx | Премия.xlsx |

## ⚙️ Настройки

Настройки автоматически сохраняются в файл `settings.json` при закрытии приложения:

- Пути к Excel-файлу и папкам
- Аккаунт Outlook
- Количество потоков
- Задержка между письмами (SMTP)
- Текст письма
- Режим отправки (Outlook/SMTP)

**Тема письма НЕ сохраняется** — генерируется автоматически при каждом запуске (прошлый месяц).

## 🔧 Возможности

- ✅ **Два режима отправки**: Outlook COM и SMTP
- ✅ **Многопоточная рассылка** (настраиваемое количество потоков, рекомендуется 1–3 для SMTP)
- ✅ **Warm-up режим** — первые 20 писем отправляются медленнее для «разогрева» аккаунта
- ✅ **Пакетная отправка** — паузы после каждых 50 писем для снижения риска блокировок
- ✅ **Jitter (случайная задержка)** — ±0.3 сек для защиты от спам-фильтров
- ✅ **Автоматическая тема письма** с прошлым месяцем
- ✅ **Прикрепление файлов** из трёх разных папок
- ✅ **Предварительный просмотр** письма перед рассылкой
- ✅ **Пауза / продолжение** рассылки
- ✅ **Отмена рассылки** в любой момент
- ✅ **Просмотр ошибок** — кнопка «📋 Ошибки» открывает файл с email-адресами неудачных отправок
- ✅ **Повторная отправка** при ошибках (SMTP)
- ✅ **Сохранение настроек** между запусками
- ✅ **Подробное логирование** всех операций
- ✅ **Копирование текста** из лога (Ctrl+C, Ctrl+Shift+C, контекстное меню)
- ✅ **Статистика в реальном времени**: отправлено, ошибок, в очереди, в процессе
- ✅ **Таймер рассылки** — отображение elapsed time в ЧЧ:ММ:СС

## ⚠️ Важные замечания

1. **Режим Outlook COM** требует установленный Microsoft Outlook
2. **Режим SMTP** работает без Outlook (прямое подключение к серверу)
3. **Только Windows** — используется COM-автоматизация для Outlook
4. **Первый запуск** может занять больше времени (инициализация Outlook)
5. **Антивирус** может блокировать EXE-файл (добавьте в исключения)
6. **Лимиты SMTP** — для Mail.ru рекомендуется:
   - 3 потока
   - Задержка 1.5 сек между письмами
   - Warm-up: 20 писем с задержкой 2.5 сек
   - Пауза 10 сек после каждых 50 писем
   - **Время рассылки 350 писем:** ~12-15 минут
7. **Jitter** — случайный разброс задержки ±0.3 сек для защиты от спам-фильтров

## 📞 Поддержка

При возникновении проблем:

1. Проверьте, что Outlook установлен и настроен (для режима Outlook)
2. Убедитесь, что Excel-файл содержит требуемые колонки
3. Проверьте логи в файле `email_sender.log`
4. Для SMTP: проверьте логин/пароль и настройки сервера
