# EmailSender - Почтовая рассылка расчетных листов ФитоФарм

Приложение для автоматической рассылки писем с вложениями через **Microsoft Outlook** или **SMTP**.

## 🚀 Быстрый старт

### Запуск из исходного кода

```bash
# 1. Установите зависимости
python -m pip install -r requirements.txt

# 2. Запустите приложение
python main.py
```

### Запуск готового EXE файла

```bash
# EXE файл находится в папке dist
dist\EmailSender.exe
```

## 📋 Требования

- **Python 3.9+** (для запуска из исходного кода)
- **Windows 7+** (требуется для работы с Outlook через COM)
- **Microsoft Outlook** (обязательно для режима Outlook COM)
- **SMTP сервер** (для режима SMTP — Gmail, Yandex, Mail.ru и др.)


После сборки EXE файл будет в папке `dist\EmailSender.exe` (~35 МБ).

## 📖 Использование

### Режим Outlook COM

1. **Запустите приложение** (`python main.py` или `dist\EmailSender.exe`)

2. **Выберите аккаунт Outlook** из выпадающего списка

3. **Укажите путь к Excel файлу** со списком получателей

4. **Укажите пути к папкам:**
   - Папка с расчетными листами
   - Папка с реестрами выдачи
   - Папка с дополнительными файлами

5. **Настройте письмо:**
   - Тема автоматически генерируется с прошлым месяцем (например, "Расчетные листы Февраль 2026")
   - Текст письма можно отредактировать

6. **Нажмите "Начать рассылку"**

7. **Отслеживайте прогресс** в окне логов

### Режим SMTP

1. **Откройте настройки SMTP** (кнопка рядом с выбором аккаунта)

2. **Заполните параметры:**
   - SMTP сервер (например, `smtp.gmail.com`)
   - Порт (465 для SSL или 587 для TLS)
   - Email (логин)
   - Пароль приложения

3. **Протестируйте подключение** кнопкой "Тест подключения"

4. **Сохраните настройки**

5. **Выберите режим отправки SMTP** и запустите рассылку

> 📋 Подробная инструкция по настройке SMTP в [`SMTP_SETUP.md`](SMTP_SETUP.md)

## 📊 Формат Excel файла

Excel файл должен содержать следующие колонки:

| Колонка | Описание |
|---------|----------|
| `email` | Адрес получателя (обязательно) |
| `файл_01` | Имя файла из папки 1 (расчетные листы) |
| `файл_02` | Имя файла из папки 2 (реестры выдачи) |
| `файл_03` | Имя файла из папки 3 (доп. файлы) |

**Пример:**

| email | файл_01 | файл_02 | файл_03 |
|-------|---------|---------|---------|
| ivanov@company.com | Иванов_ЛС.pdf | Реестр_001.xlsx | |
| petrov@company.com | Петров_ЛС.pdf | Реестр_001.xlsx | Премия.xlsx |

## ⚙️ Настройки

Настройки автоматически сохраняются в файл `config/settings.json` при закрытии приложения:

- Пути к Excel файлу и папкам
- Аккаунт Outlook / SMTP конфигурация
- Количество потоков
- Задержка между письмами (SMTP)
- Текст письма

**Тема письма НЕ сохраняется** — генерируется автоматически при каждом запуске (прошлый месяц).

## ⚠️ Важные замечания

### Для режима Outlook COM:
1. **Требуется установленный Microsoft Outlook** — без него приложение не работает
2. **Только Windows** — используется COM автоматизация
3. **Первый запуск** может занять больше времени (инициализация Outlook)

### Для режима SMTP:
1. **Требуется пароль приложения** — не основной пароль от почты
2. **Соблюдайте лимиты отправки** — см. [`SMTP_SETUP.md`](SMTP_SETUP.md)
3. **Рекомендуется задержка 1-2 сек** между письмами

### Общие:
- **Антивирус** может блокировать EXE файл (добавьте в исключения)
- **Проверяйте логи** в файле `email_sender.log` при ошибках

## 📞 Поддержка

При возникновении проблем:
1. Проверьте, что Outlook установлен и настроен (для режима Outlook)
2. Убедитесь, что Excel файл содержит требуемые колонки
3. Проверьте логи в файле `email_sender.log`
4. Для SMTP — см. [`SMTP_SETUP.md`](SMTP_SETUP.md)
