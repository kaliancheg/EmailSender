# Инструкция по сборке в EXE

## Быстрая сборка

Просто запустите:

```bash
build.bat
```

Или вручную:

```bash
# 1. Установите PyInstaller (если ещё не установлен)
python -m pip install pyinstaller

# 2. Запустите сборку
pyinstaller --clean email_sender.spec
```

## Результат

После успешной сборки EXE файл будет в папке:
```
dist\EmailSender.exe
```

## Параметры сборки

В файле `build_config\email_sender.spec` можно изменить:

- `name='EmailSender'` — имя исполняемого файла
- `console=False` — скрыть консольное окно (True для отладки)
- `icon=None` — добавить иконку (указать путь к .ico файлу)
- `upx=True` — сжатие exe (уменьшает размер)

## Добавление иконки

1. Создайте или скачайте иконку в формате `.ico`
2. Положите в корень проекта (например, `icon.ico`)
3. В `build_config\email_sender.spec` измените:
   ```python
   icon='icon.ico',
   ```

## Устранение проблем

### Ошибка "ModuleNotFoundError"
Добавьте модуль в `hiddenimports` в spec файле.

### Большой размер EXE
- Включите `upx=True` (сжатие)
- Убедитесь, что установлены только нужные зависимости

### Антивирус блокирует
- Добавьте в исключения
- Попробуйте `upx=False` (иногда UPX триггерит антивирусы)

### Ошибка с tkinter
Убедитесь, что Python установлен с tkinter:
```bash
python -c "import tkinter; print('OK')"
```

## Размер файла

Примерный размер после сборки:
- Без UPX: ~50-70 МБ
- С UPX: ~15-25 МБ

## Распространение

В папке `dist` будет:
- `EmailSender.exe` — единственный файл для распространения
- Все зависимости встроены внутрь

**Требования к системе:**
- Windows 7+
- Установленный Outlook (для работы с почтой)
