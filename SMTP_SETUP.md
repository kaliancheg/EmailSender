# 📧 Настройка SMTP для отправки писем

## Преимущества SMTP перед Outlook COM

| Характеристика | Outlook COM | SMTP |
|---------------|-------------|------|
| Запрос подтверждения | ❌ Требуется | ✅ Нет |
| Требуется Outlook | ❌ Да | ✅ Нет |
| Контроль очереди | ❌ Ограничен | ✅ Полный |
| Логирование | ❌ Базовое | ✅ Подробное |
| Повторная отправка | ❌ Нет | ✅ Есть |
| Работа в фоне | ❌ Нет | ✅ Да |

---

## 📋 Необходимые настройки

### Обязательные параметры:

| Параметр | Описание | Пример |
|----------|----------|--------|
| **SMTP сервер** | Адрес SMTP сервера | `smtp.gmail.com` |
| **Порт** | Порт подключения | `465` (SSL) или `587` (TLS) |
| **Email (логин)** | Ваш email адрес | `user@gmail.com` |
| **Пароль приложения** | Специальный пароль | `abcd efgh ijkl mnop` |
| **SSL/TLS** | Шифрование | Зависит от порта |

---

## 🔐 Пароль приложения

**Важно:** Для большинства почтовых сервисов нужно создать **пароль приложения**:

### Gmail:
1. Перейдите в [Аккаунт Google](https://myaccount.google.com/)
2. Безопасность → Двухэтапная аутентификация
3. Пароли приложений → Создать
4. Скопируйте 16-значный пароль

### Yandex:
1. Перейдите в [Пароли приложений](https://passport.yandex.ru/profile-app-passwords)
2. Создать новый пароль
3. Выберите "Почта"
4. Скопируйте пароль

### Mail.ru:
1. Перейдите в [Настройки безопасности](https://account.mail.ru/security)
2. Пароли для внешних приложений
3. Создать пароль
4. Скопируйте пароль

### Outlook/Hotmail:
1. Перейдите в [Безопасность Microsoft](https://account.microsoft.com/security)
2. Дополнительные параметры безопасности
3. Создать новый пароль приложения

## 📊 Отслеживание очереди писем

### Статусы писем:

| Статус | Описание |
|--------|----------|
| 🕐 **В очереди** (pending) | Ожидает отправки |
| 📤 **Отправляется** (sending) | В процессе отправки |
| ✅ **Отправлено** (sent) | Успешно отправлено |
| ❌ **Ошибка** (failed) | Не удалось отправить |
| 🔄 **Повтор** (retry) | Попытка повтора |

### Статистика рассылки:

- **Всего писем** — общее количество
- **Отправлено** — успешные отправки
- **Ошибка** — неудачные попытки
- **В очереди** — ещё не отправлены
- **Процент выполнения** — прогресс бар
- **Процент успеха** — соотношение успешных к завершённым

## 🔧 Использование в приложении

### 1. Откройте настройки SMTP

В главном окне нажмите **"Настройки SMTP"** (кнопка добавляется рядом с выбором аккаунта)

### 2. Заполните параметры

- Выберите быстрый пресет (Gmail/Yandex/Mail.ru/Outlook)
- Или введите свои параметры вручную
- Нажмите **"Тест подключения"** для проверки
- Нажмите **"Сохранить"**

### 3. Запустите рассылку

- Выберите режим отправки: **Outlook** или **SMTP**
- Настройте остальные параметры
- Нажмите **"Начать рассылку"**

## 🛠️ Устранение проблем

### Ошибка авторизации
- Проверьте логин и пароль
- Убедитесь, что используете **пароль приложения**, а не основной пароль
- Проверьте, включена ли двухфакторная аутентификация

### Ошибка подключения
- Проверьте SMTP сервер и порт
- Убедитесь, что брандмауэр не блокирует порт
- Попробуйте переключить SSL/TLS

### Письма не отправляются
- Проверьте лимиты отправки почтового сервиса
- Убедитесь, что аккаунт не заблокирован
- Проверьте логи приложения

## 💡 Рекомендации

1. **Начинайте с малых объёмов** — протестируйте на 5-10 письмах
2. **Используйте паузы** — добавляйте задержки между письмами (1-2 секунды)
3. **Мониторьте логи** — отслеживайте ошибки и повторы
4. **Сохраняйте настройки** — приложение запоминает SMTP конфигурацию
